GAUDVIBE PORTFOLIO - TECHNOLOGIES USED
=======================================

FRONTEND CORE
-------------
• HTML5 - Document structure and semantics
• CSS3 - Styling, animations, and responsive layouts
• JavaScript (ES6+) - Interactive functionality and DOM manipulation

WEBGL & GRAPHICS
----------------
• WebGL API - Hardware-accelerated graphics rendering
• GLSL (OpenGL Shading Language) - Custom fragment shader programming
  - Signed Distance Functions (SDF) for shape rendering
  - Real-time color palette generation
  - Distortion effects and animations
  - Dynamic pattern generation
• Canvas API - Pixel manipulation and color sampling

JAVASCRIPT FEATURES
-------------------
• Object-Oriented Programming (Class-based architecture)
• Modern ES6+ Syntax:
  - Arrow functions
  - Template literals
  - Destructuring assignment
  - Event delegation
• Animation loop with requestAnimationFrame (60fps)
• Touch event handling (touchstart, touchend)
• Dynamic DOM element creation and styling

CSS & DESIGN
------------
• CSS Grid - 2x2 button grid layout
• CSS Animations and Keyframes:
  - Floating arrow animation (hover)
  - Ripple effect on click
  - Smooth transitions
• Responsive Design with media queries
  - Breakpoints: 768px, 600px, 480px
• Custom typography: Courier New monospace
• Multi-layered box shadows for depth effect
• Backdrop filter for blur effects
• Mobile-specific optimizations

PERFORMANCE
-----------
• Efficient WebGL rendering with minimal draw calls
• Debounced window resize handlers
• Optimized animation loops
• CSS hardware acceleration
• Fallback for non-WebGL browsers

INTERACTIVITY
-------------
• Hover effects with animated visual feedback
• Click/touch ripple animations
• Transparent button backgrounds on interaction
• Real-time color sampling from shader
• External link handling with smooth transitions

RESPONSIVE FEATURES
-------------------
• Mobile-first approach
• Touch device detection (hover: none, pointer: coarse)
• Adaptive animation amplitudes
• Flexible grid layouts
• Optimized touch targets

BROWSER SUPPORT
---------------
• Modern browsers with WebGL support
• Graceful degradation for older browsers
• Touch-optimized interactions
• Responsive across all screen sizes
